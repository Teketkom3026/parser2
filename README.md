# Parser2 — Парсер контактной информации

Автоматизированный сбор контактных данных с веб-сайтов: ФИО, должности, email, телефоны, ИНН/КПП.

## Возможности

- **Режим 1** — Парсинг по целевым должностям (фильтрация по топ-менеджменту и т.д.)
- **Режим 2** — Сбор всех контактов сотрудников
- **Быстрый старт** — Один URL, мгновенный результат
- Экспорт в XLSX с нормализацией данных
- WebSocket прогресс в реальном времени
- Blacklist по доменам и email
- Поддержка robots.txt

## Стек

- **Backend**: FastAPI + Playwright (headless Chromium) + SQLite
- **Frontend**: React 18 + TypeScript + Vite
- **Export**: openpyxl (нормализация без AI)

## Запуск (Docker)

```bash
cp .env.example .env
docker-compose up --build -d
```

Веб-интерфейс: `http://SERVER_IP/parser2/`

## Структура выгрузки XLSX

| Колонка | Описание |
|---------|----------|
| Название компании | Из og:site_name / title |
| Сайт (домен) | Источник |
| Общий email | Корпоративный email |
| Общий телефон | Телефон компании |
| ФИО | Извлечено regex/DOM |
| Должность (как на сайте) | Сырые данные |
| Должность (нормализованная) | После нормализации |
| Категория роли | Топ/Средний менеджмент / Специалист |
| Личный email | email сотрудника |
| Личный телефон | телефон сотрудника |
| ИНН / КПП | Регексом из страниц |
| Соцсети | VK, LinkedIn, Telegram и др. |
| URL страницы-источника | Откуда взяты данные |
| Язык страницы | ru/en/... |
| Дата сканирования | Дата обработки |
| Статус / Комментарий | OK / Частично / Ошибка |
